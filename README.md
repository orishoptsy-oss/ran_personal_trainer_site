# רן נהוראי - אתר תדמית מאמן כושר אישי

## 🏋️ תיאור הפרויקט

אתר תדמית מקצועי ויוקרתי למאמן כושר אישי **רן נהוראי**. האתר מציג את השירותים, התוצאות, והביקורות של רן בעיצוב מודרני ומושך.

כרגע האתר משתמש בתמונות דוגמה. כדי להוסיף תמונות אמיתיות:

1. העלה תמונות ל-GitHub (בתיקייה `images/`)
